# mobileconfig.sh
Скрипт для генерации `.mobileconfig` файлов для iOS/macOS VPN на базе strongSwan.

## Настройка переменных

Перед запуском скрипта отредактируйте переменные в начале файла:

| Переменная | Описание | Пример |
|------------|----------|--------|
| `CLIENT` | Имя клиента (должно совпадать с именами файлов сертификатов) | `me` |
| `SERVER` | Отображаемое имя VPN в системе | `"StrongSwan VPN"` |
| `FQDN` | IP-адрес или домен VPN-сервера | `"185.103.255.86"` |
| `CA` | Имя CA (должно совпадать с файлом CA сертификата) | `"ca"` |

## Настройка автоматического подключения

В скрипте есть параметр OnDemandEnabled, который управляет автоматическим подключением VPN на iOS/macOS.
Для его активации, поменяйте значение с 0 на 1:
```xml
<key>OnDemandEnabled</key>
<integer>0</integer>
```

## Расположение сертификатов на сервере

Скрипт ожидает, что сертификаты на сервере лежат в стандартных директориях strongSwan:

```bash
# Закрытый ключ клиента
/etc/ipsec.d/private/${CLIENT}.pem

# Сертификат клиента
/etc/ipsec.d/certs/${CLIENT}.pem

# Корневой сертификат CA
/etc/ipsec.d/cacerts/${CA}.pem
```
